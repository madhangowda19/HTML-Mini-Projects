HTML

Definition

HTML (HyperText Markup Language) is the standard markup language used to create and design web pages. It structures the content of a webpage by using elements (called tags) that tell the browser how to display text, images, links, and other multimedia.

Basic HTML Tags

  1. Block-level Tags
    • <html> – Root element
    • <head> – Metadata container
    • <body> – Main content area
    • <div> – Generic block container
    • <p> – Paragraph
    • <h1> to <h6> – Headings
    • <ul> – Unordered list
    • <ol> – Ordered list
    • <li> – List item
    • <table> – Table container
  2. Inline Tags
    • <a> – Hyperlink
    • <img> – Image
    • <span> – Inline container
    • <br> – Line break
    • <strong> – Important text (bold)
    • <em> – Emphasized text (italic)
    • <b> – Bold text
    • <i> – Italic text
    • <u> – Underlined text
    • <mark> – Highlighted text

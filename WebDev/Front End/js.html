Javascript

Definition

JavaScript (JS) is a high-level, interpreted programming language that enables dynamic behavior on web pages. It allows you to manipulate HTML elements, handle events, validate forms, create animations, interact with APIs, and build complex web applications. JS works in the browser as well as on servers (Node.js).

JavaScript Features Classification

  1. Core Language Constructs
    • var / let / const – Declaring variables
    • function – Defining reusable functions
    • if / else – Conditional statements
    • switch – Multiple condition handling
    • for / while / do-while – Loops
    • break / continue – Loop control
    • return – Returning a value from a function
    • try / catch / finally – Error handling
    • typeof – Checking data type
    • console.log() – Printing output to console
  2. DOM & Browser Interaction
    • document.getElementById() – Access element by ID
    • document.getElementsByClassName() – Access elements by class
    • document.querySelector() – Select first matching element
    • document.querySelectorAll() – Select all matching elements
    • element.innerHTML – Get or set HTML content
    • element.style – Change inline CSS styles
    • element.addEventListener() – Attach events to elements
    • alert() – Display popup alert
    • prompt() – Ask user input
    • setTimeout() / setInterval() – Time-based operations

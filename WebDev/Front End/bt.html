BootStrap

Definition

Bootstrap is a popular open-source front-end framework used for designing responsive and mobile-first websites. It provides ready-to-use CSS classes, components, and JavaScript plugins that help developers quickly create modern, consistent, and visually appealing web pages without writing all CSS and JS from scratch.

Most Used Bootstrap Components / Classes

  1. Layout & Structure
    • container / container-fluid – Fixed-width or full-width container
    • row – Creates a horizontal group of columns
    • col / col-{size}-{number} – Defines responsive columns
    • d-flex – Enables flexbox layout
    • justify-content-* – Horizontal alignment in flex containers
    • align-items-* – Vertical alignment in flex containers
    • card – Flexible content container with header, body, footer
    • navbar / navbar-expand – Navigation bar
    • jumbotron / bg-* – Highlighted content area or background color
    • list-group / list-group-item – Styled list groups
  2. UI Components & Utilities
    • btn / btn-primary / btn-secondary – Buttons with different styles
    • form-control – Styled input fields
    • input-group – Grouped inputs with add-ons
    • alert / alert-success / alert-danger – Notification messages
    • badge / badge-pill – Small count or label indicators
    • modal – Popup dialog box
    • dropdown / dropdown-menu / dropdown-item – Dropdown menus
    • table / table-striped / table-bordered – Styled tables
    • img-fluid – Responsive images
    • text-center / text-left / text-right – Text alignment utility classes
